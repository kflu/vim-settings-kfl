===================
KFL's vim settings
===================

How to use
==========
Go to your home path, clone the repo::

    git clone --recursive git@github.com:puzzl3r/vim-settings-kfl.git

or, as read-only::

    git clone --recursive git://github.com/puzzl3r/vim-settings-kfl.git

Then cd into the repo, execute the ``install.bat`` with admin permission.
